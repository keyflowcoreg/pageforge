# PageForge

AI landing page generator powered by Gemini 2.0 Flash.

[![Live Demo](https://img.shields.io/badge/Live_Demo-pageforge.vercel.app-blue?style=for-the-badge)](https://pageforge-phi.vercel.app)

[Live Demo](https://pageforge-phi.vercel.app) | [Report Bug](https://github.com/keyflowcoreg/pageforge/issues)

## Features

- Describe your business, get a complete landing page in seconds
- Powered by Gemini 2.0 Flash for fast, high-quality generation
- 5 template styles with live iframe preview
- Device-responsive preview (desktop, tablet, mobile)
- Export as clean Next.js code or standalone HTML

## Tech Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · Gemini 2.0 Flash · Framer Motion · Stripe + x402 Payments

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:4007](http://localhost:4007) to view it locally.
